## Naming Conventions

- **Files and folders:** lowercase, ,  nouns, snake_case (`my_script.py`, `data/`, `outputs/`)  
- **Scripts:** action-oriented verbs (`watch_list_screener.py`)  
- **Data files:** descriptive nouns (`sp500_companies.csv`)  

Consistency helps with readability, maintainability, and cross-platform compatibility.
